Course Summary

Setting Objectives

1.1 It's very important to set your objectives for the landing page right from the start - Free

1.2 The objectives have to be realistic, easy to remember and measurable in order to work

1.3 Sketch the layout, elements and add the main Call To Action on paper in your notepad

Content Creation

2.1 Create the landing page layout in a graphic design application and add elements - Free

2.2 Use the layout preview created in your favorite image editing program to add content

2.3 Find quality images and create videos that explain what your landing page is all about

HTML Coding

3.1 Find a good HTML landing page template as foundation for your online project - Free

3.2 Edit the code using HTML, CSS, Bootstrap and add images using an image editing app

3.3 Upload the project to a live server and make sure your hosting provider is reliable
